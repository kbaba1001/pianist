# PIANIST

PIANIST is a graceful keybind.

## Installation

then execute:

    $ wget https://raw.githubusercontent.com/kbaba1001/pianist/master/dot.Xmodmap -O ~/.Xmodmap
    $ xmodmap ~/.Xmodmap
