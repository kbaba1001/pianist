# PIANIST

PIANIST is a graceful keybind.

## Installation

then execute:

    $ git clone git@github.com:kbaba1001/pianist.git ~/.pianist
    $ cp ~/.pianist/dot.Xmodmap ~/.Xmodmap
    $ xmodmap ~/.Xmodmap
